APIについて

  • YouTube Player APIでYouTubeの動画を表示方法を制御出来ます。
  • 動画を変更する場合はapp.jsの10行目のvideoIdを変更して下さい。赤字部分　videoId: 'qyTBc1SIDmI'
  • YouTube JavaScript Player API リファレンスの詳細はこちら
  • YouTube Player APIを使用しない場合はこのページを参考に画像を挿入してください。

  • Flickr APIでFlickrの画像を検索してサイトに埋め込むことが出来ます。
  • Flickr APIを使用する場合はapp.jsの24行目のapi_keyを必ずご自身のものと変更してください。
  • 赤字部分　"api_key": "2ea7c3bea55a2ae28f53cb6099451f78",
  • api_keyはこちらから取得できます。
  • 検索語を変更する場合はapp.jsの26行目を変更してください。赤字部分　"text": "music",　
  • Flickr APIを使用しない場合はこのページを参考に画像を挿入してください。
